# Frontend (Next.js Application) Specification - Project SOVEREIGNTY

**Status:** TBD (Awaiting User Feedback) | **Version:** 2.0

## 1. Overview
The Frontend design and architecture are currently pending based on feedback from initial prototype testing.

## 2. Core Goals
- **Personalization:** Deliver tailored content and calls to action based on the **Unified User Identity** (PSAI + Donor + CRM data).
- **Site-Wide Semantic Search:** A unified search bar allowing natural language discovery across all content types.
- **Multimedia Integration:** Seamless display of transcripts, video (YouTube), and audio (SoundCloud).
- **Phyllis AI (The Oracle):** A sophisticated chat interface for interacting with the archival materials.

## 3. Tech Stack (Preliminary)
- **Framework:** Next.js (App Router).
- **Styling:** Tailwind CSS.
- **State Management:** TanStack Query + Zustand.
